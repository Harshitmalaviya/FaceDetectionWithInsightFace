# Face Detection With InsightFace
Used Face detection for classification of images based on phot library. 
In this Python notebook, Face detection using InsightFace library used. To store the metadata, we used FAISS Vector database. CV2 for read-write images and InsightsFace for Face analysis with metadata like below face embeddings, age, gender etc.

You can find more details:
https://github.com/deepinsight/insightface
